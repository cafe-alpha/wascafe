First revision, by hitomi2500, Summer 2015

Specs :
 - MAX10 FPGA (10M04SA or higher)
 - 32MB SDRAM support
 - UART + USB interface

Features :
 - Cartridge backup memory (volative)
 - SD (not SDHC) card access
 - RAM expansion (untested ?)
